Font Style

CSS dapat mengatur style dari teks yang ada pada halaman web yang dibuat. Terdapat beberapa property yang dapat digunakan untuk mengatur style teks yaitu sebagai berikut

  • text-align: dighunakan untuk mengatur perataan teks, value yang dapat digunakan yaitu left, center, right
  • color: dighunakan untuk memberikan warna pada text
  • letter-spacing: mengatur jarak antar karakter
  • word-spacing: dighunakan untuk mengatur jarak antar kata.
